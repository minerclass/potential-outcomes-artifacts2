# Research Outcomes & Action Studio

An interactive, GitHub Pages-ready public artifact for exploring possible dissertation outcomes, responsible community responses, and evidence-to-action decisions connected to:

*Pedagogical Friction in the Age of Generative AI and Tertiary Algorithmicity.*

This studio is prospective. It demonstrates possible findings and responsible decision processes; it does not present hypothetical scenarios as completed dissertation findings.

## What the studio includes

- **Orientation:** Evidence-to-action chain, guiding commitments, and entry routes.
- **Scenario Lab:** Eight possible outcome scenarios, with mixed-methods relationship controls for convergence, expansion, divergence, and silence.
- **Ecological Map:** Interactive Bronfenbrenner-inspired map connecting classroom, relationships, district systems, culture, and time.
- **Practice-to-Policy:** Six-phase intervention cycle with local review progress.
- **Decision Lab:** A proportionate-response builder based on evidence condition, risk, reversibility, privacy, and equity.
- **Dissemination:** Audience-specific products, boundaries, purposes, and feedback loops.
- **Stewardship:** An interactive public/de-identified/private research-data classification activity.

## GitHub Pages setup

No build step or dependency installation is required.

The included `.github/workflows/pages.yml` workflow deploys the repository as a static GitHub Pages site whenever `main` is updated. If Pages has not previously been enabled, open **Settings > Pages** once and select **GitHub Actions** as the source.

After deployment, verify the public page before sharing.

## Local use

Open `index.html` directly in a browser. Local progress for the Practice-to-Policy Cycle is stored only in the browser through `localStorage`.

## Suggested ecosystem role

This works best as one cohesive repository because the modules share a common evidence-to-action logic. It can later link from:

- the Dissertation Sites Dashboard as a research-facing planning and findings-translation artifact;
- the Writing Sites Hub as a public scholarship and practitioner-facing artifact;
- the Proposal Defense Studio as the prospective answer to, "What will you do with the findings?"

If a module grows into a substantial independent resource, the strongest candidates for separate repositories are:

- the **Practice-to-Policy Cycle** as a district professional-learning tool;
- the **Ecological Map** as a public explainer;
- the **Decision Lab** as a policy and governance protocol.
